# showcase_cal

> [calendly](https://calendly.com/) clone in Deno

# Development

First copy `.env.example` to `.env` and replace `CLIENT_ID` and `CLIENT_SECRET`
to appropriate values.

Then run the deployment locally:

```sh
deno task dev
```
